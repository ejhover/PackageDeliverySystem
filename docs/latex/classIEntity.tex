\doxysubsection{Detailed Description}
Represents an entity in a physical system. 

An \doxylink{classIEntity}{IEntity} object has a unique ID, a position, a direction, a destination, and details. It also has a speed, which determines how fast the entity moves in the physical system. Subclasses of \doxylink{classIEntity}{IEntity} can override the {\ttfamily Update} function to implement their own movement behavior.
